What is Habiliti?

An Introduction

Habiliti allows you to track existing or new habits in an easy to use interface.

Does it work on my device?

Habiliti works on most desktop browsers, but not all mobile browsers.

Features

Current Features

At the moment, the current list of features include:

  • Tracking multiple habits
  • Enabling or disabling notifications
  • Saving your data to a local database

Future Features

Here is the proposed list of future features:

  • Ability to edit existing habits
  • Creating push notifications
  • Faster retrieval of data
